To Do List

Example of a simple task list. The user has a text box to type the task name and can add it to the list by clicking the "Create" button. Once added, the task can be marked as completed and, if desired, deleted later. Tasks are saved in the browser, allowing them to be automatically loaded the next time the user accesses the page, unless the cache is cleared. The project was made using html/css/js as a base.

    • Finantial Control

    Simple web application for financial management, developed in React. The application allows users to create cards to organize and track their finances. After creating the cards, users can add items, record gains and expenses, and view the final balance of each card throughout the process. Users can work with multiple cards if needed.     • Pokedex

    Example of a simple Pokédex, featuring a total of 1025 Pokémon retrieved from a web-based API. The data is organized across 7 table pages, allowing users to search for Pokémon by their IDs or names. As users switch between pages, the corresponding Pokémon are dynamically loaded into the table. Page and filter settings are saved in the browser, ensuring these preferences are automatically restored on the next visit, unless the user clears the cache. The project was made using html/css/js as a base.

    • PokeAPI

    Example of an API developed in C# .NET using the CQRS architecture, with MongoDB as the chosen database. This is an API for Pokémon integration, designed for use in a user and Pokémon registration system. The API implements flows for user registration, user deletion, password change, login verification, refresh token, identifying Pokémon linked to the user, and updating the user's Pokémon list. Additionally, the API includes unit tests to ensure the integrity and proper functioning of the services.

    Calculator

    Example of a simple calculator developed in C# .NET using Windows Forms and the CuoreUI graphical framework. The calculator stores numerical and operational inputs as text, executing the operation when the result button is pressed. It also retains the last result and provides an option to clear the text box when needed. Additionally, the interface components are automatically adjusted if the user resizes the window.

    Analogical Watcher

    Example of a simple analog clock developed in C# .NET using Windows Forms. The clock displays the time with precision to the second, automatically adjusting to the local time zone. The entire graphical interface, including the hour markers and hands, was created programmatically, along with the animated behavior of the hands.
